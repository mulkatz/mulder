# Mulder Demo v2

Separate UI prototype for the cleaner, technical Mulder workbench direction.

## Commands

```bash
npm install
npm run dev
npm run build
npm run preview
```

`npm run dev` and `npm run preview` use port `5174`.

## Design Tokens

The v2 look is controlled from `src/styles.css`.

- Surface tokens: `--canvas`, `--panel`, `--panel-raised`, `--field`
- Text tokens: `--text`, `--text-muted`, `--text-subtle`, `--text-inverse`
- Structure tokens: `--border`, `--border-strong`, `--ring`
- Accent tokens: `--accent`, `--accent-hover`, `--accent-soft`
- Status tokens: `--success`, `--warning`, `--danger`, `--info`
- Density tokens: `--sidebar-width`, `--topbar-height`, `--radius-*`

This prototype intentionally avoids the v1 editorial serif language. It is sans-first, compact, light-first, and optimized for dense analysis screens.
